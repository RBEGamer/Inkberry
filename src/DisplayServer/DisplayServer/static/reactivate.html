MANAGE YOUR INKBERRY

This link enables you to manage the current dispay render state. Its possible to activate or deactivate the screen, so no further updates will be rendered on the screen until enabled again.




The delete option removes the display from the system. If the current display is a parent display, all child displays will be deleted too. A configuration backup is stored in the database for later manual restore. All displays can be setup again, using their QR codes.
