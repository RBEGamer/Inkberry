About INKBERRY

Inkberry is an innovative startup specializing in the development of customizable digital door signs. Using state-of-the-art technology, Inkberry enables the simple and stylish display of information directly on the door. Our solutions offer flexible customization options, intuitive operation and an elegant design that blends seamlessly into any environment. Whether in the office, hotel or at home, Inkberry's digital door signs set new standards in information and communication.

IMPRESSUM

HARDWARE

Our digital door signs use the latest E-INK technology to operate for a long time without electricity. Thanks to integrated solar cells, it is possible to guarantee battery-free operation! This means that there are no additional operating costs.
Our software platform also supports many common open-source E-INK displays.

SOFTWARE

Inkberry provides an open and innovative web-based software platform to deliver content to our hardware displays. Sophisticated automation ensures efficient operation without additional user intervention. A plugin-based component system also allows the user to easily develop their own plugins, so any data source can be used with the system.
